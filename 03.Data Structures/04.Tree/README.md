# 트리(Tree)

트리는 연결 리스트와 같이 노드 기반 자료 구조이지만 트리의 **여러** 노드로의 링크를 포함할 수 있다.

![alt text](image.png)

- 가장 상위 노드 (사진에서는 1)를 **루트(root)** 라 부른다. \
사진에서의 루트는 보다시피 트리의 **꼭대기** 다. 전형적으로 트리를 묘사하는 방식이다.

- 사진에서 "1"을 "2"와 "3"의 **부모** 라 한다. 반대로 "2"와 "3"은 "1"의 **자식**이다.

- 패밀리 트리에는 노드에 **자손**과 **조상**이 있을수 있다. 한 노드의 자손은 그 노드에서 생겨난 모든 노드이며, 한 노드의 조상은 그 노드를 생겨나게 한 모든 노드이다.

- 트리에는 **레벨**이 있다. 각 레벨은 트리에서 같은 줄(row)이다. 사진의 트리는 세 레벨이다.

- 트리의 프로퍼티는(property)는 **균형 잡힌**정도를 말한다. \
모든 노드에서 하위 트리의 노드 개수가 같으면 그 트리는 **균형 트리**다. \
사진의 트리는 3레벨의 우측 노드가 하나 없으므로 **불균형 트리다**.
