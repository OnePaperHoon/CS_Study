# CS_Study

이 레포지토리는 개인적인 컴퓨터 공학/CS 공부 노트를 정리한 저장소입니다. 각 폴더에 개념 설명, 예제 코드, 빌드 및 실행 방법을 정리해 두었습니다.

주요 폴더

- `Algorithms/` : 정렬, 탐색 등 알고리즘 설명과 예제 코드 (C)
	- `Binary Search/`, `Bubble Sort/` 등
- `Data Structures/` : 링크드 리스트, 동적 배열 등 자료구조 구현 예제
	- `dynamic_array/` : C로 구현한 동적 배열 예제
- `Database/` : 데이터베이스 기초 노트 및 실습 자료
- `Operating Systems/` : 운영체제 개념 및 실습
- `Computer Networks/` : 네트워크 개념 정리
- `Computer Architecture/` : 컴퓨터 구조 관련 정리
- `template/` : 예제용 `Makefile` 등 템플릿

사용 방법

- 각 예제 폴더에 `Makefile`이 존재하면, 해당 폴더에서 `make`로 빌드하세요.
- 소스 코드나 설명을 추가할 때는 관련 폴더의 `README.md`를 함께 업데이트해 주세요.

기여

- 개선사항이나 추가 예제가 있으면 PR로 제안해 주세요. 간단한 설명과 실행 방법을 포함하면 리뷰가 빨라집니다.
