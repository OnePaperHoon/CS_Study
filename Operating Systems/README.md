	 # Operating Systems

	운영체제 관련 개념들을 정리하는 폴더입니다. 프로세스/스레드 관리, 메모리 관리(가상 메모리, 페이지 교체), 파일 시스템, 입출력 스케줄링, 동기화(뮤텍스/세마포어) 등을 다룹니다.

	권장 구성

	- 개념 정리(예: 프로세스 vs 스레드, 컨텍스트 스위칭)
	- 간단한 실습 코드(스레드 동기화 예제 등)
	- 문제 풀이 및 요약 노트

	예시: 스레드 동기화 테스트

	```powershell
	cd "Operating Systems"
	# 스레드/동기화 예제 컴파일 및 실행(존재 시)
	make
	.
	```
